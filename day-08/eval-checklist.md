# Prompt Evaluation Checklist

1. Does the prompt clearly state the task? Yes/No
2. Does the prompt specify the audience? Yes/No
3. Does the prompt define the exact output format? Yes/No
4. Does the prompt give context needed to answer correctly? Yes/No
5. Does the prompt avoid vague words like "better" or "nice"? Yes/No
6. Does the prompt include an example when the format is non-obvious? Yes/No
7. Does the prompt set a length/scope limit (word count, number of items)? Yes/No
8. Does the prompt avoid asking multiple unrelated questions at once? Yes/No
9. Would two different people interpret this prompt the same way? Yes/No
10. Does the prompt specify tone (formal, casual, technical)? Yes/No
