# Day 8 – Reflection

**Patterns in good prompts:**
1. They name the exact output format (bullet points, table, Yes/No + reason) instead of leaving structure to guess.
2. They limit scope (word count, number of items, specific factors to consider) so the model doesn't wander.
3. They specify the audience or context, which shapes tone and depth automatically without extra instructions.

**Patterns in bad prompts:**
1. They use vague verbs like "make it better" or "is this suspicious" with no defined criteria for what "better" or "suspicious" means.
2. They leave output format undefined, leading to inconsistent length/structure every time.
3. They assume the model has context it was never actually given (e.g. "classify this" without saying which categories exist).
